# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly.

**Do NOT open a public issue.**

Instead, email: **roger@homeatix.net**

You will receive an acknowledgement within 48 hours and a detailed response within 7 days.

## Scope

This repository is a reproducible self-publishing build kit. Security concerns include:

- Accidental exposure of proprietary business data
- Personally identifiable information (PII)
- Credentials, tokens, or API keys
- Internal knowledge graph or investor materials

## Disclosure Policy

- We will investigate all reports promptly
- We will not take legal action against good-faith reporters
- We will credit reporters (with permission) in any advisory

## Security Controls

- `.gitignore` enforced for all business-sensitive files
- Git history audited and purged as of 3 April 2026
- W-Anchor Protocol provides build integrity verification
- No secrets, tokens, or credentials stored in this repository
