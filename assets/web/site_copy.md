# Reproducible SelfPub Kit

A practical, automation-first workflow for building and shipping clean EPUB + print-ready PDFs from Markdown.

## What you get
- One-command builds (PDF, EPUB, DOCX)
- Reproducible outputs (same inputs  same artifacts)
- A reviewer-ready package you can zip and send

## Who this is for
Authors writing technical books, docs, or code manuals who want fewer tools and fewer surprises.
